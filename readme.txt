== Description ==

Bstone Light add extra features to [Bstone](https://wpbstone.com/) like widgets, activate/desactivate the customizer sections and demo import.
This plugin requires the [Bstone](https://wpbstone.com/) theme to be installed.

= Attribution =

This plugin Uses WP Bstone Demos API to import demo data.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bstone-light` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but it does not work =

This plugin will only work with the [Bstone](https://wpbstone.com/) theme.
